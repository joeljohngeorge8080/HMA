---

## Role-Based Navigation Visibility

### CEO
Sees all modules.

### Heads
Sees all modules.

### HR
Sees all modules.

### Finance
Sees all modules.

### Project Officer
Sees:
- Dashboard
- Projects
- Reports & Analysis
- Audit Logs

Does not see:
- Staff & Payroll
- Attendance
- Expense Management
- Finance
- Settings

---

## Navigation Flow

### Login Flow
Login Screen
→ Dashboard

### Dashboard Flow
Dashboard
→ Projects
→ Staff & Payroll
→ Attendance
→ Expense Management
→ Finance
→ Reports & Analysis
→ Audit Logs

### Projects Flow
Projects
→ Category View
→ Project List
→ Project Detail Page
→ Actual Expense Section
→ Reports

### Staff Flow
Staff & Payroll
→ Employee List
→ Employee Detail Drawer
→ Salary History
→ Payroll Generation

### Attendance Flow
Attendance
→ Monthly View
→ Machine Import
→ Excel Import
→ Correction Log

### Expense Flow
Expense Management
→ Expense List
→ Upload Expense Excel
→ Edit Expense
→ Delete Expense
→ Analysis

### Finance Flow
Finance
→ Finance Excel Upload
→ Finance Records
→ File History
→ Reports

### Reports Flow
Reports & Analysis
→ Attendance Report
→ Project Report
→ Overall Project Report
→ Forecast Report
→ Predicted Report
→ Actual Expense Report
→ Export

### Audit Flow
Audit Logs
→ Filter Logs
→ Open Log Details
→ View History
