# Excel Import Specifications

## Attendance Import

**Source:** Pace Attendance Software

**File Type:** Excel (.xlsx)

### Columns
- Employee ID
- Employee Name
- Date
- Status
- In Time
- Out Time
- Work Duration
- Late By
- Early By
- Overtime
- Shift

### Validation Rules
- Employee ID must exist.
- Date cannot be empty.
- Status must be valid.
- Duplicate records are not allowed.
